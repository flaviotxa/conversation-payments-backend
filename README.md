To be able to run the app you need to create a `.env` file and specify in this file the connection url for MongoDB, for example:

```
MONGO_URL=mongodb://localhost:27017/development
TWILIO_ACCOUNT_SID=...
TWILIO_AUTH_TOKEN=...
STRIPE_PUBLISHABLE_KEY=
STRIPE_SECRET_KEY=
SUCCESS_URL=
CANCEL_URL=
```
